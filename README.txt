1. Download the tesseract installer and install it in Program Files/Tesseract-OCR
   Get tesseract from here: http://www.mediafire.com/file/oi8m6n9ipmmsz1a/tesseract-ocr-w64-setup-v5.0.0-alpha.20201127.exe/file

2. Download eng.traineddata and replace the original eng.traineddata in Program Files/Tesseract-OCR/tessdata
   Get eng.traineddata from here: https://drive.google.com/file/d/1RLVlj_RljTBwb_UzPkDWdgeIlWF40hGN/view?usp=sharing

3. Run pip install pytesseract in cmd.

4. Run pip install pdfminer in cmd.
